>**3.-Días laborales**

En este ejercicio se crea un fichero con los días festivos del año 2017. No se cuentan los días del verano como festivos puesto que la mayoría de la gente trabaja. En este ejercicio se introduce un rango de fechas mediante el datePickerDialog se comprueba que el rango sea correcto y que no se hayan introducido las fechas. En este programa primero sacamos el margen de ambas fechas pasadas y después se comprueba que el dia no sea festivo, si no lo es este se Imprime en el EDITTEXT. Se imprimen los días laborales.

Mencionar también que a la hora de seleccionar la fecha con el datePickerDialog he tenido un problema y es que se selecciona un año incorrecto (Creo recoradar que es el 3917 en el caso de seleccionar el 2017) el ejercicio se realiza de forma correcta pero este imprime una fecha erronea.

>**5.-Descarga de Imágenes**

Se lee un fichero que esta en el servidor donde nos encontramos la ruta a las imagenes a mostrar. Este lee el fichero y guarda los string de las rutas de las imagenes en un Array de String. Cada vez que pulsamos el botón de siguiente este lee la posición del array correspondiente y avanza (De forma circular, de la ultima pasa a la primera) o retrocede una posición. Al descargar se selecciona la posición siguiente a la actual.
